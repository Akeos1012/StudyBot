**Asymptotic Analysis** - A method in computer science used to evaluate how efficient an algorithm is as the input size becomes very large.

- focuses on how the **time (speed)**
- focuses on how the **space (memory)**

Requirements of an algorithm grow, rather than exact performance on a specific machine.

## Why it is used?

It helps compare algorithms in a way that is independent of hardware, programming language, or small input differences.

**Common notations:**
- **Big O (O)** → worst-case growth
- **Omega (Ω)** → best-case growth
- **Theta (Θ)** → average/exact growth

In short, asymptotic analysis is a way to measure and compare algorithm performance based on how it scales with large amounts of data.
