**Edge Computing** – Edge computing is a **computing model where data is processed closer to where it is generated (the “edge” of the network), instead of sending everything to a centralized cloud or data center**. 

- Reduces delay by processing data near [[Device]] or [[Sensor]] sources
- Minimizes reliance on [[Cloud Computing]] for real-time tasks
- Improves speed and responsiveness (low [[Latency]])
- Reduces network traffic and bandwidth usage
- Used in real-time systems like IoT, autonomous vehicles, and smart devices
- Often works together with [[Cloud Computing]] for storage and heavy processing

**Example:**
- Smart cameras processing video locally instead of sending everything to the cloud
- Self-driving cars analyzing road data in real time
- Smart factories making instant machine decisions

#### How System Works?

Data Source → Edge Device (Local Processing) → Immediate Response → Optional Cloud Storage/Analysis
