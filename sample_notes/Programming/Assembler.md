**[[Assembler]]** – A **software tool that converts assembly language (low-level human-readable instructions) into machine code (binary instructions) that the CPU can execute directly**. It acts as a translator between assembly code and hardware-level execution.

- Converts [[Assembly]] into [[Machine Code]] (binary)
- Produces executable instructions that run on the [[CPU]]
- Works closely with [[Instruction Set Architecture]] (ISA)
- Translates low-level symbolic instructions into hardware operations
- Part of the software development toolchain for low-level programming
- Typically used in system programming, embedded systems, and firmware

An assembler enables direct control over hardware by translating human-readable low-level instructions into binary code that the processor understands.
