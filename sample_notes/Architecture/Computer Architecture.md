**Computer Architecture** – Computer architecture is the **design structure and conceptual framework of a computer system that defines how its components (CPU, memory, and input/output systems) work together to execute instructions**.

- Defines how the [[CPU]] processes instructions
- Explains how [[Memory]] and storage interact with the processor
- Describes how data moves through [[Bus System]] and components
- Determines system performance and efficiency
- Includes instruction design and execution methods
- Works closely with [[Instruction Set Architecture]] (ISA)

**Examples:**
- Modern Intel and AMD processors (x86 CISC-based)
- ARM processors in smartphones (RISC-based)
- Microcontrollers in embedded systems
- Gaming consoles CPU/GPU designs
- Supercomputer architectures

___
### Types of Computer Architecture

- **Von Neumann Architecture** → shared memory for data and instructions
- **Harvard Architecture** → separate memory for data and instructions
- **Modified Harvard Architecture** → hybrid approach used in modern CPUs
- **RISC Architecture** → simplified instruction set for efficiency
- **CISC Architecture** → complex instruction set for powerful operations

___

In short, Computer Architecture is the structural design and organization of a computer system that defines how its components interact to process data and execute instructions.
