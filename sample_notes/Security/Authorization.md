**Authorization** – The **process of determining what an authenticated user is allowed to do or access within a system**.

- Happens after [[Authentication]]
- Controls access to [[Data]], [[Software]], and system resources
- Uses roles and permissions (e.g., admin, user, guest)
- Common in Security systems and [[Access Control]]

In short, Authorization is the process of defining and enforcing what an authenticated user is permitted to access or perform in a system.
