**Data Analysis** – Data analysis is the **process of inspecting, cleaning, transforming, and interpreting data to discover useful information, patterns, and insights that support decision-making**.

- Involves collecting and organizing raw [[Data]]
- Uses statistical methods, tools, and algorithms to find patterns
- Helps identify trends, correlations, and anomalies
- Commonly used in [[Business Intelligence]] and reporting
- Supports decision-making in fields like business, science, and technology
- Often performed using tools like [[Excel]], [[Python]], and [[SQL]]
- Works closely with Data Visualization to present insights clearly

Data analysis is the process of turning raw data into meaningful insights that can be used for decisions and problem-solving.
#### How System Works?

Raw Data → Cleaning → Processing → Analysis → Insights → Visualization → Decision

In short, data analysis is the process of examining raw data to extract meaningful insights that help in decision-making and problem-solving.
