**Logical Operations** – Operations that manipulate and evaluate **logical values (TRUE/FALSE or 1/0)** to make decisions and control program execution.

Inside the **[[ALU]]**, logical operations compare and combine binary values to determine outcomes.
### Common logical operations:

- **AND** → TRUE only if all conditions are TRUE
- **OR** → TRUE if at least one condition is TRUE
- **NOT** → reverses a value (TRUE ↔ FALSE)
- **XOR** → TRUE when inputs are different

**Example:**
- 1 AND 1 → 1
- 1 AND 0 → 0
- 1 OR 0 → 1
- NOT 1 → 0

#### How computers use it:

- [[IF statements]]
- [[Conditional branching]]
- Search and filtering
- Program decision-making
- Digital circuits and [[Logic Gates]]

In short, Logical operations are instructions that let computers evaluate conditions and make decisions using binary values.
