**Algorithm** - A <u>step-by-step set of instructions</u> used to <u>solve a problem or complete a task</u>.

In computer science, it’s the **Logic or Method** a computer follows before it actually becomes code. An algorithm can tell a computer how to:

- perform calculations
- sort [[Data]]
- make decisions
- process information

Algorithm = Clears, finite sequence of steps to solve a problem. 

## Difference of Algorithm vs Program

| <center></center>Metric | Algorithm                                                    | Program                                     |
| ----------------------- | ------------------------------------------------------------ | ------------------------------------------- |
| Nature                  | Abstract concept or logical plan                             | Concrete, physical implementation           |
| Language                | Plain English, diagrams, or [[Pseudocode]]                   | Specific [[Programming Languages ]]         |
| Execution               | Interpreted and analyzed by humans                           | Executed directly by a computer machine     |
| Analysis                | Measured via [[Asymptotic Analysis]] (Time/Space Complexity) | Evaluated via dynamic [[Software ]] testing |
#### Conceptual Level (How problems are defined)

This level focuses on how problems are understood, structured, and converted into logical steps before any coding begins.

- Problem Solving
- [[Logic]]
- Computational Thinking
- [[Flowchart]]
- [[Pseudocode]]

#### Implementation Level (How algorithms become programs)

This level represents how logical steps are translated into executable instructions using programming tools and languages.

- [[Programming Languages]]
- [[Compiler]]
- [[Interpreter]]
- [[Source Code]]
- [[Software]]

#### Data Processing Level (What algorithms operate on)

This level focuses on how data is organized, stored, and manipulated during algorithm execution.

- [[Data]]
- [[Dataset]]
- [[Data Structure]]
- [[Array]]
- [[Linked List]]

#### Performance Analysis Level (How efficiency is measured)

This level evaluates how fast and efficient an algorithm is in terms of time and memory usage.

- [[Asymptotic Analysis]]
- [[Time Complexity]]
- [[Space Complexity]]
- [[Big O Notation]]

#### Execution Level (How algorithms run in systems)

This level explains how algorithms are processed and executed inside computer systems at runtime.

- [[CPU]]
- Process
- [[Memory Management]]
- [[Runtime Environment]]

In short, Algorithms move from conceptual thinking, to implementation, to data handling, to performance evaluation, and finally execution inside computing systems.
