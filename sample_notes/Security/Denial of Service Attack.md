**Denial of Service Attack** – (DoS) an attack is a **cyberattack that attempts to make a computer system, website, server, or network unavailable to legitimate users by overwhelming it with excessive requests or exhausting its resources**.

#### How it Works?

- Attacker sends huge amounts of traffic or requests
- Target system becomes overloaded
- Resources (CPU, memory, bandwidth) get exhausted
- Legitimate users cannot access the service

**Example:**

Normal traffic:

```
100 users → Website → Works
```

Attack traffic:

```
1,000,000 fake requests → Website → Slow / Crash
```

___
#### Common forms

- **Traffic Flooding** → overwhelming bandwidth
- **Request Flooding** → excessive server requests
- **Application-layer attacks** → targeting application resources
- **Distributed Denial of Service (DDoS)** → attack comes from many devices at once

In short, Denial of Service (DoS) Attack is a cyberattack that overloads a system or service with excessive requests, making it unavailable to legitimate users.
