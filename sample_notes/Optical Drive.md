**Optical Drive** – An optical drive is a hardware device that uses a **laser** to read and/or write data on optical discs such as CDs, DVDs, and Blu-ray discs

An optical drive is a device that reads and writes data using laser technology on discs like CDs and DVDs.

- Reads data from optical discs
- Writes (burns) data onto blank discs
- Installs software or games from discs
- Plays media like music and movies
### Types of optical drives

- **[[CD Drive]]** → reads/writes CDs
- **[[DVD Drive]]** → reads/writes DVDs and CDs
- **[[Blu-ray Drive]]** → supports Blu-ray, DVDs, and CDs (high capacity)
- **[[Read-only Drive]]** → can only read discs
- **[[RW Drive (Read/Write)]]** → can read and burn data

#### How it works? 

Disc → Laser in Optical Drive → Data Read/Write → Computer System

The laser reads tiny pits and lands on the disc surface to interpret binary data.

#### Why it is Used?

- Installing older software and operating systems
- Watching movies and listening to music
- Archiving and backing up data
- Physical media distribution

In short, an optical drive is a device that uses laser technology to read and write data on CDs, DVDs, and Blu-ray discs.
