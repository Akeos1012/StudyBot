**[[Compiler]]** – A **software tool that translates high-level programming language code into machine code (binary instructions) that a computer’s CPU can execute directly**. It processes the entire program at once, checks for errors, and converts it into an executable format.

- Translates [[Programming Languages]] like C, C++, or Java into machine code
- Converts source code into executable files that the [[CPU]] can run
- Performs error checking during compilation (syntax and semantic errors)
- Optimizes code for better performance and efficiency
- Works as part of the software development toolchain
- Produces output that runs directly on the operating system or hardware

A compiler acts as a bridge between human-readable code and machine-level instructions, enabling software to run on computer hardware.
