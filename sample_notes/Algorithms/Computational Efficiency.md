**Computational Efficiency** – Computational efficiency is the **measure of how effectively an algorithm or program uses computational resources such as time (CPU) and memory while solving a problem**.

- Measures how much [[Time Complexity]] an algorithm requires
- Measures how much [[Space Complexity]] (memory) is used
- Focuses on reducing unnecessary operations in code
- Improves performance of [[Algorithms]] and software systems
- Important in large-scale systems and real-time applications
- Directly affects speed, scalability, and system load

Computational efficiency is the study of how well an algorithm performs in terms of resource usage like time and memory.

**Example:**
- Searching using a hash table → fast and efficient
- Searching using a linear scan → slower and less efficient

#### How System Works?

Problem → Algorithm Design → Execution → Resource Usage Measured → Optimize for Less Time & Memory

In short, Computational Efficiency is the measure of how effectively an algorithm or program uses time and memory resources to solve a problem while minimizing waste and maximizing performance.
