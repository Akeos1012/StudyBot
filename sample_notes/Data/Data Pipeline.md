**Data Pipeline** – A data pipeline is a **set of processes and tools that move, transform, and process data from one system to another in an automated and structured way**, usually from raw data sources to storage, analysis, or applications.

- Moves [[Data]] from sources (databases, APIs, logs) to destinations
- Performs steps like cleaning, transforming, and validating data
- Often used in Data Engineering and Data Science workflows
- Can be batch-based (processed in chunks) or streaming (real-time)
- Common in [[Cloud Computing]] environments and big data systems
- Works with tools like [[ETL]] (Extract, Transform, Load)

It is a structured process for moving, transforming, and preparing data across systems for storage and analysis.
