Comparison Operations - These are operations the CPU uses to **compare two values** and determine their relationship. They are handled inside the **[[ALU]]** and are used to make decisions in programs.
#### Common comparison operations

- `==` → equal to
- `!=` → not equal to
- `>` → greater than
- `<` → less than
- `>=` → greater than or equal to
- `<=` → less than or equal to
#### How the CPU uses it

**Comparison operations are used in:**
- [[IF statements]]
- [[Loops]] (while, for)
- decision-making logic in programs

**Example:** 
IF (5 > 3) → TRUE → execute code block

In short, comparison operations are [[CPU]] instructions that compare values and return a [[Boolean]] result used for decision-making in programs.
