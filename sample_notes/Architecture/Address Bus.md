**Address Bus** – The address bus is a **set of physical or logical lines in a computer system used by the [[CPU]] to specify the memory location where data should be read from or written to**.

It does not carry the actual data. Instead, it carries the _address_ of where the data is located in Memory or [[Storage Devices]]. The address bus tells the system _where_ data should be accessed in memory.

- Identifies memory locations for read/write operations
- Works alongside [[Data Bus]] and [[Control Bus]]
- Enables communication between [[CPU]] and [[RAM]]
- Determines how much memory a system can directly access
#### How it Works? 

CPU → Address Bus → Memory Location → Data Bus → CPU
