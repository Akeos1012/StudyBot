**Distributed Operating System** – An **operating system that manages a group of independent computers and makes them work together as a single unified system**.

- Connects multiple [[Computer]] systems
- Shares Resources ([[CPU]], memory, storage) across machines
- Coordinates tasks across a [[Network]]
- Handles communication between systems
- Provides users a single-system view

#### How it works?

Instead of one machine doing everything:

```
Computer A + Computer B + Computer C→ Work together as one system
```

The OS manages:

- task distribution
- synchronization
- data sharing
- fault handling

### Where it is used?

- [[Cloud Computing]] systems
- [[Cluster Computing]]
- [[Supercomputers]]
- Large enterprise servers
- Distributed databases

#### Simple analogy

A distributed OS is like:

multiple workers in different rooms being controlled by one manager who makes them act like a single team instead of separate people.

In short, A Distributed Operating System is an operating system that manages multiple connected computers and makes them function as a single unified system by sharing resources and coordinating tasks across a network.
