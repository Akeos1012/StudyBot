**Wi-Fi** – A **wireless networking technology** that allows devices to connect to the internet or local networks using radio signals instead of physical cables. Wi-Fi is a technology that enables wireless communication between devices and a network (like the internet).

- Connects devices to the internet without wires
- Transfers data using [[Radio Waves]]
- Allows multiple devices to share the same network
- Enables communication between devices on a [[Local Network]]

- [[Router]] → sends and receives internet signals
- [[NIC]] → enables device connection
- [[Internet]] → global network of data

#### Where it is Used?

- Smartphones
- Laptops
- Smart TVs
- Gaming consoles
- [[IoT]] devices (smart home devices)

| Wi-Fi                 | Ethernet          |
| --------------------- | ----------------- |
| Wireless              | Wired             |
| Uses radio signals    | Uses cables       |
| More flexible         | More stable       |
| Can have interference | Less interference |

In short, Wi-Fi is a wireless technology that connects devices to a network and the internet using radio signals instead of physical cables.
