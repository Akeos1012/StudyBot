**BIOS** – The BIOS is low-level firmware stored on the **[[Motherboard]]** that runs when a computer is first powered on. It initializes hardware components and starts the process of loading the operating system.

- Checks hardware during startup (Power-On Self Test / POST)
- Initializes components like **[[CPU]]**, **[[RAM]]**, and storage
- Finds and loads the operating system into memory
- Provides basic system settings and configuration

#### How it works?

Power ON → BIOS starts → Hardware check → Load OS → System runs

#### BIOS vs UEFI

Modern systems often use [[UEFI]] instead of traditional BIOS:

**BIOS** → older, simpler firmware
**UEFI** → modern version with faster boot, better security, and graphical interface

#### Why is it Important?

- Without BIOS/UEFI, the computer cannot start
- Acts as the bridge between hardware and operating system
- Ensures system hardware is working before OS loads

In short, BIOS is firmware that starts the computer, checks hardware, and loads the operating system so the system can run properly.
