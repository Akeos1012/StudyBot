**Cloud Computing** – Cloud computing refers to the **delivery of computing services such as storage, processing power, databases, networking, and software over the internet**, instead of relying on local physical hardware. It allows users and organizations to access and use computing resources on-demand from remote servers.

- Provides computing services over the [[Internet]] instead of local machines
- Uses remote [[Server]] infrastructure to process and store data
- Supports virtualization to allocate resources efficiently
- Enables scalable and on-demand access to computing power
- Reduces the need for maintaining physical hardware locally

Cloud computing is a system architecture that shifts computing workload from local devices to remote infrastructure managed by providers, enabling global access to shared computing resources.
