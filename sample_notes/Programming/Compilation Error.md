**Compilation Error** - A **type of programming error that occurs when source code cannot be successfully translated (compiled) into machine code because it violates the rules of the programming language**.

#### Why it happens?

A compiler checks code before it runs. If something breaks the rules:

- missing symbols
- wrong syntax
- undefined variables
- type mismatches
- incorrect structure

Then it stops the process.

**Example:**
```
int x = "hello";
```

**Problem:**
- you are trying to store text in an integer

**Result:**
- compilation error

#### Common causes

- [[Syntax Error]]
- [[Type Error]]
- missing semicolons
- undeclared variables
- incorrect function calls
- mismatched brackets


#### Simple analogy
It’s like:

- writing an essay in a language the translator doesn’t understand
- so it refuses to translate anything at all

In short, a Compilation Error is a programming error that occurs when source code violates language rules and cannot be successfully compiled into executable machine code.
