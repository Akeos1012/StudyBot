**ASCII** - **(American Standard Code for Information Interchange)** A character [[Encode]] system that represents letters, numbers, symbols, and control characters as numerical values so computers can store and process text.

Each character is assigned a unique number, which is then converted into **[[Binary Code]] (0s and 1s)** for [[Computer]] use.

In short, ASCII is one of the earliest standards that allowed <u>computers to represent human-readable text using numbers</u>.
