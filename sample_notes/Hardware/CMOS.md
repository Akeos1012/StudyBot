**[[CMOS]] (Complementary Metal-Oxide-Semiconductor)** – CMOS refers to a **technology used to build low-power integrated circuits and a small memory area on the motherboard that stores system configuration settings such as BIOS/UEFI settings and system time**. In computing context, “CMOS” often refers specifically to the small memory chip that retains critical system information even when the computer is powered off.

- Stores system configuration data like [[BIOS]] / [[UEFI]] settings
- Maintains real-time clock information (date and time)
- Powered by the [[CMOS Battery]] when the system is off
- Located on the [[Motherboard]] as part of firmware support hardware
- Uses low-power semiconductor technology for efficient data retention
- Works closely with the [[Boot Process]] during system startup

CMOS is both a semiconductor manufacturing technology and a practical system component used for storing essential firmware settings in modern computers.

#### How System Works?

In short, CMOS is a low-power memory technology used in computers to store essential system settings and time information that persists even when the computer is turned off.
