**Binary code** - A system of representing using only two symbols: 0 & 1. 

It is the fundamental language of computers, where all **text, data, and instructions** are encoded using combinations of binary digits. A **[[Bit]]** represents a single binary value (0 or 1), and multiple bits are grouped into a **[[Bytes]]** to form meaningful data patterns.

Binary works because computers use electronic states that can be interpreted as:

- **Off = 0**
- **On = 1**

This system is based on the idea of logical true/false values, originally formalized by **George Boole’s Boolean algebra**, which later became the foundation for digital computing systems.

### **How Binary Translates to Text?** 

[[Computer]] rely on agreed-upon encoding systems like [[ASCII]] or Unicode. 

When you type "Hi". The computer will convert it into binary using standard ASCII.

Example: "Hi"

- The letter **"H"** translates to `01001000`
- The letter **"i"** translates to `01101001`

### **From Binary to Number Systems**

Binary is just one type of [[Numeral Systems]] used to represent data. Computers don’t naturally understand letters or symbols. They only understand numbers in different base systems.

The 4 Numeral Systems:

- **Binary (Base-2)** → used internally by computers
- **[[Decimal]] (Base-10)** → used by humans
- **[[Octal]] (Base-8)** → shorthand for binary
- **[[Hexadecimal]] (Base-16)** → compact way to represent binary data
