**Pseudocode** -  **simple, informal way of writing a program’s logic using plain language mixed with programming structure**, without following the strict syntax of any programming language. Pseudocode is a step-by-step written plan of an algorithm that looks like code but is not real programming code.

- Planning program logic before coding
- Designing [[Algorithms]]
- Explaining how a program works
- Communicating ideas between programmers

#### Why it is useful?

- Helps organize thinking before writing real code
- Reduces errors in programming logic
- Makes complex problems easier to break down
- Acts as a bridge between human thinking and actual code

In short, Pseudocode is a structured way of describing programming logic in plain language before converting it into real code.
