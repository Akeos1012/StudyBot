**Data Type** – A data type is a **classification that defines what kind of value a variable can hold and what operations can be performed on it in programming**.
### Common data types

- **Integer (int)** → whole numbers (e.g., 1, 50, -10)
- **Float/Double** → decimal numbers (e.g., 3.14, 99.5)
- **Character (char)** → single letter or symbol (e.g., 'A')
- **String** → text (e.g., "Hello")
- **Boolean (bool)** → true or false

#### Why it is important?

- Helps [[Programming]] languages store and process data correctly
- Prevents errors like mixing text and numbers
- Defines memory usage in [[Computer]] systems
- Enables correct Operations (math, comparison, logic)
- Used in [[Algorithms]] design and [[Data Structure]]

In short, A Data Type is a classification in programming that defines what kind of value a variable can store and what operations can be performed on it.
