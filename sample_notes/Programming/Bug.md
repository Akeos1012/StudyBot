**Bug** – is an **error, flaw, or unintended behavior in a computer program that causes it to produce incorrect results or behave differently than expected**.

Yes, it’s basically your code refusing to behave the way you _very confidently told it to behave_.

- Occurs in [[Software]] and [[Programming]] code
- Can cause crashes, wrong outputs, or unexpected behavior
- Often introduced during Development or logic design mistakes
- Found and fixed using a [[Debugger]]
- Can appear in [[Algorithms]], [[Data Structure]] usage, or system logic
- Fixed through a process called **Debugging**

A bug is a defect in software that prevents it from functioning as intended.

#### Why bugs happen

- Human mistakes (most common, unfortunately)
- Wrong assumptions in logic
- Unexpected input
- Poor design of [[Algorithms]]
- Complex system interactions
