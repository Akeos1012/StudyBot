**Chipset** – The chipset is a group of electronic components on the **[[Motherboard]]** that manages communication between the **[[CPU]]**, **[[RAM]]**, **[[Storage Devices]]**, and other peripherals.

It acts as the **control center of data flow** between the processor and the rest of the system.

- Coordinates communication between CPU and other components
- Manages data flow between RAM, storage, and peripherals
- Controls input/output operations
- Determines hardware compatibility and system features

#### Main parts of a chipset: 

**Northbridge (older systems)** → handled fast communication (CPU, RAM, GPU)
**Southbridge (older systems)** → handled slower devices (USB, storage, audio)

#### Why is it Important?

- Enables all hardware components to work together
- Defines system capabilities (USB ports, PCIe lanes, storage support)
- Affects overall performance and expandability

In short, The chipset is the motherboard’s control system that manages communication between the CPU and all other hardware components.

Chipset = traffic system directing data flow
