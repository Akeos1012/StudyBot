**Big Data** - Refers to extremely large, complex, and continuously growing datasets that are too massive or fast-moving to be processed effectively using traditional data processing systems.

It is not just about “large size,” but also about the speed, variety, and complexity of data being generated and analyzed. Big Data is data that is too large and complex to be handled by normal data systems.

### Core characteristics (often called the “V’s”)

- Volume → extremely large amount of data
- Velocity → fast generation and processing of data
- Variety → different types of data (text, images, video, logs)
- Veracity → accuracy and reliability of data
- Value → useful insights extracted from data
