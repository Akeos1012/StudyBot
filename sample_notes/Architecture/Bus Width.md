**Bus Width** – Bus width refers to the **number of bits that can be transmitted simultaneously through a system bus in a single operation**. It determines how much data can be transferred at once between components such as the [[CPU]], Memory, and other hardware units.

It represents a key hardware-level specification that directly affects system performance, since a wider bus allows more data to move in parallel, improving communication speed between system components.

Bus width is a property of the computer’s [[System Bus]] architecture that defines the capacity of data transfer channels used inside the system.

#### How System Works?

Data Source → [[CPU]] / Memory → System Bus → Data Transfer (limited by Bus Width) → Destination Component

In short, Bus Width is the number of bits that can be transferred at once through a system bus, directly affecting how much data the system can move between components in a single operation.
