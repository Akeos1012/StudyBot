**Browser** – A **software application used to access, retrieve, and display information from the [[World Wide Web]]**.

It acts as an interface between the user and web content by sending requests to web servers and rendering the response into readable pages like text, images, videos, and interactive elements. A browser is a program that lets you view and interact with websites on the internet.

- Sends requests to web servers using [[HTTP]] or [[HTTPS]]
- Receives data like HTML, CSS, JavaScript
- Renders web pages visually for users
- Manages tabs, history, and bookmarks
- Executes client-side scripts
#### How it Works? 

User → Browser → HTTP/HTTPS Request → Web Server → Response → Rendered Web Page
