**[[CSV]]**  – (Comma-Separated Values) A **simple file format used to store and exchange tabular data (like spreadsheets and databases) in plain text, where each line represents a row and values are separated by commas**.

- Stores structured [[Data]] in plain text format
- Each line represents a row in a table
- Values are separated by commas (or sometimes other delimiters like semicolons)
- Commonly used for importing/exporting data between [[Spreadsheet]], [[Database]], and applications
- Easy to read, write, and process using tools like [[Python]], [[Excel]], and [[SQL]]
- Lightweight and widely supported across systems

CSV is a basic data format used to represent tabular information in a simple, readable text structure that can be shared across many platforms.

### CSV file content:

```
Name,Age,Course
Alice,20,Computer Science
Bob,22,Information Technology
Charlie,19,Engineering
```

## What it represents as a table:

|Name|Age|Course|
|---|---|---|
|Alice|20|Computer Science|
|Bob|22|Information Technology|
|Charlie|19|Engineering|

In short, CSV is a simple plain-text file format used to store and exchange tabular data using comma-separated values.
