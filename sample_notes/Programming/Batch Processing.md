**Batch Processing** - A **method of executing a group of tasks or data jobs together as a single unit without continuous user interaction during execution**.

![[Pasted image 20260622220317.png]]

Instead of processing inputs one by one in real time, the system collects data, groups it, and processes it in batches at scheduled times or when resources are available. Batch processing is when tasks are collected and executed together as a group instead of one at a time.

- Data or tasks are collected first
- They are grouped into a batch
- The system processes them automatically
- Results are produced after completion, not instantly

#### How it Works?

Data Collection → Job Queue → Batch Execution → Output Generation → Storage/Result

In short, Batch processing is a computing method where multiple tasks are collected and executed together automatically without real-time user interaction, improving efficiency in large-scale systems.
