**Motherboard** – The **main circuit board of a computer** that connects and allows communication between all internal hardware components.

![[Pasted image 20260617232418.png]]

It acts as the **central hub** of the computer, enabling components to exchange data, receive power, and work together as one system.

- Connects all hardware components
- Transfers data between components
- Distributes electrical power
- Supports communication through buses and connectors
- Provides interfaces for input and output devices

 **Main components connected to the motherboard:**
- [[CPU]]
- [[RAM]]
- [[Storage Devices]]
- [[GPU]]
- [[PSU]]
- [[Expansion Cards]]
- [[NIC]]

In short, the motherboard is the central circuit board that connects, powers, and coordinates all computer hardware components.
