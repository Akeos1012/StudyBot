Conditional Branching - A concept in programming and computer architecture where a system **chooses different paths of execution based on a condition**.

**A [[Computer]] checks a condition (TRUE or FALSE), then:**
- If TRUE → do one thing
- If FALSE → do something else

**Basic flow:**
- IF condition = TRUE → go to Instruction A  
- IF condition = FALSE → go to Instruction B
#### Where it is used?

- [[IF statements]]
- [[Else statements]]
- [[Loops]] (WHILE, FOR)
- decision-making in [[AI]]
- game logic
- [[Operating System]] processes

In short, conditional branching is the process where a computer chooses different execution paths based on whether a condition is true or false, allowing programs to make decisions.
