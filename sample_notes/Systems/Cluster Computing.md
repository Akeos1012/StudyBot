**[[Cluster Computing]]** – Cluster computing refers to a **computing model where multiple independent computers (nodes) are connected and work together as a single unified system to perform tasks, share workloads, and improve performance, reliability, and scalability**. These nodes are typically connected through a high-speed network and coordinated to function like one powerful system.

- Combines multiple [[Server]] machines into a single coordinated system
- Uses [[Distributed Systems]] to split and manage workloads across nodes
- Improves performance through parallel execution using [[Parallel Processing]]
- Increases reliability through redundancy and fault tolerance
- Commonly used in [[Cloud Computing]] and large-scale data processing systems
- Connected via high-speed [[Networking]] infrastructure for communication

Cluster computing is a system design approach that groups multiple computers together to work as a unified computing resource, often used in high-performance computing, cloud platforms, and enterprise systems.
