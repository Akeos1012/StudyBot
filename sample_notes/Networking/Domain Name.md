**Domain Name** –  A **human-readable address used to identify and access websites and internet services instead of typing numerical IP addresses**. DNS translates the domain into the correct server address behind the scenes.

- Identifies websites and internet services
- Converts names into [[IP Address]] through [[DNS (Domain Name System)]]
- Makes navigation easier for users
- Supports websites, email, and applications
- Creates a recognizable online identity

**Example:**

```
Domain Name → google.com
IP Address  → 142.250.xxx.xxx
```

You type:

```
google.com
```

DNS translates it into the server’s IP and connects you.

#### Parts of a Domain

# 🧩 Parts of a domain

```
www.example.com
│   │        └── Top-Level Domain (TLD)
│   └──────────── Second-Level Domain 
└──────── Subdomain
```

Example:

- **www** → [[Subdomain]]
- **example** → Domain Name
- **.com** → top-level domain ([[TLD]])

___
### Types of Domain Name

#### 1. Top-Level Domain (TLD)

The **rightmost part** of a domain name.

Examples:

```
.com.org.net.edu
```

Example:

```
google.com       ↑      TLD
```

#### 2. Generic Top-Level Domain (gTLD)

Domains intended for **general or global use**.

Examples:

```
.com.org.net.info.app.tech
```

Used by:

- businesses
- organizations
- public websites

#### 3. Country Code Top-Level Domain (ccTLD)

Domains assigned to **countries or regions**.

Examples:

```
.ph → Philippines.jp → Japan.uk → United Kingdom.de → Germany
```

Used for geographic identity and local services.

#### 4. Sponsored Top-Level Domain (sTLD)

Domains with **restricted use managed by specific organizations**.

Examples:

```
.edu.gov.mil.museum
```

Registration usually follows eligibility rules.

#### 5. Second-Level Domain (SLD)

The **registered name** directly before the TLD.

Example:

```
google.com↑SLD
```

This is usually the brand or organization name.

#### 6. Subdomain (Third-Level Domain)

A subdivision created under a main domain.

Examples:

```
mail.google.comblog.example.comshop.company.com
```

Useful for organizing services.

In short, a Domain Name is a readable internet address that allows users to access websites and services, while DNS translates it into an IP address that computers can understand. Domain names are organized into types such as TLD, gTLD, ccTLD, sTLD, SLD, and Subdomains.
