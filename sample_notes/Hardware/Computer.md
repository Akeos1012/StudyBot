**Computer** - An electronic device that **receives data, processes it using instructions, stores it, and produces output as useful information**.

___
#### Brief History of Computer

**After [[UNIVAC I]] (1951), computers rapidly evolved:**

- **1950s–1960s** → [[Mainframe]] computers used by governments and large companies for data processing and scientific calculations.
- **1970s** → Introduction of [[microprocessors]] made computers smaller and more affordable.
- **1980s** → Personal Computers (PCs) became common in homes and offices (e.g., IBM PC, Apple Macintosh).
- **1990s** → Rise of the **[[Internet]]**, connecting computers globally and transforming communication and information access.
- **2000s – present** → Development of laptops, smartphones, cloud computing, [[AI]] systems, and high-performance computing.

___
#### Computer Hardware 

[[Computer Hardware]] refers to the **physical components of a computer system**.

**Example:** 
- [[CPU]] → processes instructions
- [[RAM]] → temporary working memory
- [[Storage Devices]] (HDD, SSD) → long-term data storage
- [[Motherboard]] → connects all components
- [[GPU]] → handles graphics processing

___
#### External Devices 

[[External Devices]] connected to a computer to extend its functionality:

- [[Input Devices]]→ keyboard, mouse, scanner
- [[Output devices]] → monitor, printer, speakers
- [[Storage Devices]] → USB drives, external hard disks

___
#### Internet 

The [[Internet]] is a **global network of connected computers** that allows:

- Communication (email, messaging, video calls)
- Information sharing ([[Website]], [[Database]])
- Online services ([[Cloud Storage]], streaming, banking, [[AI]] tools)

It transformed computers from standalone machines into **connected systems**.

___
#### How Computers Process Information

Inside a computer, billions of [[Transistor]] rapidly switch between binary states (`0` and `1`) to represent digital information.

Groups of transistors are combined into Logic Gates, which perform simple logical decisions and calculations using binary inputs.

These logic gates are then combined into larger digital circuits that build major computer components:

**Transistors:**
- [[Logic Gates]]
- [[ALU]]
- [[CPU]]

Through this layered structure, a computer can receive [[Data]], process instructions, make decisions, store results, and generate output.

This is how modern computers transform simple electrical signals into complex tasks such as running programs, displaying graphics, connecting to the [[Internet]], and executing operating systems.

___
#### Operating System 

**An [[Operating System]] (OS) is software that:**
- Manages hardware and software resources
- Allows users to interact with the computer
- Runs applications

___

In short, computer is a programmable electronic system that processes [[Data]] into information. It evolved from early machines like UNIVAC I into modern devices such as PCs, smartphones, and cloud-connected systems powered by hardware, software, and the Internet.
