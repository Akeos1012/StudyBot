**CPU Utilization** – CPU Utilization refers to the **percentage of time the CPU is actively working on executing processes instead of being idle**. It measures how efficiently the processor is being used by the operating system and running programs.

- Indicates how much of the [[CPU]] processing power is being used
- Expressed as a percentage (0% = idle, 100% = fully busy)
- Managed and influenced by [[CPU Scheduling]]
- Improves with efficient [[Process Management]]
- Affected by [[Context Switching]] and system workload
- Used to evaluate system performance and efficiency

It is a performance metric used by the operating system to measure how effectively the CPU is being used by running processes.

#### How System Works?

Processes → Scheduled by [[CPU Scheduling]] → Executed by [[CPU]] → Utilization Measured Over Time

In short, CPU utilization is a performance metric that shows how much of the CPU’s capacity is actively being used to run processes.
