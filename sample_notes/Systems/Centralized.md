**Centralized** – Centralized refers to a **system design or organizational structure where control, decision-making, and data processing are concentrated in a single central point or authority**. In this model, all major operations depend on one main system or server that manages and coordinates all connected components.

#### How System Works?

Clients / Nodes → Central Server → Processing & Control → Response back to Client

In short, centralized means a system where control and decision-making are handled by a single central authority or system that all other components depend on.
