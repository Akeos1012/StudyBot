**[[Wi-Fi Card]]** – A Wi-Fi card is a hardware component that allows a computer to **connect to wireless networks (Wi-Fi)** so it can access the internet and communicate with other devices without using cables.

A Wi-Fi card is the part of a computer that enables wireless internet connection.

- Connects the computer to Wi-Fi networks
- Sends and receives data wirelessly
- Communicates with routers and access points
- Converts data between the computer and radio signals
#### Types of Wi-Fi cards

- **Integrated Wi-Fi Card** → built into the **[[Motherboard]]** or laptop
- **PCIe Wi-Fi Card** → installed using **[[Expansion Slots (PCIe)]]**
- **USB Wi-Fi Adapter** → external plug-in device

#### How it Works? 

Computer → Wi-Fi Card → Wireless Signals → Router → Internet

The Wi-Fi card converts digital data into radio waves and back again.

**It Connects:**
- Routers
- Hotspots
- Access points
- Other wireless devices
#### Why it is Important? 

- Enables wireless internet access
- Removes need for Ethernet cables
- Allows mobility (especially for laptops and phones)
- Supports modern networking and cloud services

In short, a Wi-Fi card is a hardware component that enables a computer to connect to wireless networks and access the internet without cables.
