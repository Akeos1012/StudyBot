**NIC** – (Network Interface Card) A hardware component that allows a computer to **connect to a network or the internet** by enabling communication between the computer and other devices.

A NIC is the part of a computer that lets it send and receive data over a network.

- Connects a computer to a local network or the internet
- Sends and receives data packets
- Translates data between the computer and network cables or wireless signals
- Provides a unique hardware address called a MAC address

___
### Types of NIC

- **[[Ethernet]]** → uses cables for connection
- **[[Wi-Fi]]** → connects using wireless signals
- **Integrated NIC** → built into the **[[Motherboard]]**
- **Expansion NIC** → added through **[[Expansion Slots (PCIe)]]**

#### How it Works?

Computer → NIC → Network (Router/Internet) → Other Devices

It converts computer data into network signals and vice versa.

#### Why it is Important?

- Enables internet access
- Allows file sharing between devices
- Supports online communication (games, calls, browsing)
- Essential for all network-based systems

In short, a NIC is a hardware component that enables a computer to connect and communicate over a [[network]] or the [[internet]].
