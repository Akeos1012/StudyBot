**Else Statement** – An else statement is a **control flow structure in programming that defines what code should run when a previous condition (if statement) is false**.

In simple terms:

> It tells the program what to do when the “if condition” fails.

- Works with [[If Statement]] for decision-making
- Executes only when the condition is false
- Helps create alternative execution paths in [[Programming]]
- Common in all [[Programming Languages]] like C++, Java, Python, JavaScript
- Improves logic flow and program control
- Used in conditional branching and decision systems

An else statement is used to define an alternative block of code when a condition is not satisfied.

---
### Types of Else Usage

- **[[Else Statement]]**
- **[[Else If (elif)]]** 
- **[[Nested Else]]** 

**Examples:**
- If it is raining → take umbrella, else → don’t take umbrella
- If score ≥ 50 → pass, else → fail
- Login system → correct password vs incorrect password

In short, An else statement is a programming control structure that executes a block of code when the associated if condition is false, providing an alternative path in decision-making logic.
