**Backpropagation** – Backpropagation is a **learning algorithm used in [[Neural Network]] and [[Deep Learning]] systems to reduce errors by adjusting internal parameters based on how wrong the output is**.

It works by calculating the error at the output layer and then sending that error backward through the network to update weights and improve accuracy.

Backpropagation is the process of correcting a neural network by sending error information backward to improve learning.

- Compares predicted output vs actual output
- Calculates error (loss)
- Sends error backward through layers
- Updates [[Weights and Biases]]
- Repeats until model improves
#### How it works?

Input → Forward Pass → Output → Error Calculation → Backward Pass → Weight Update

In short, Backpropagation is the learning mechanism in neural networks that reduces prediction error by propagating mistakes backward and adjusting model parameters to improve accuracy over time.
