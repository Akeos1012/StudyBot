**Expansion Slots (PCIe)** – PCIe (Peripheral Component Interconnect Express) expansion slots are high-speed connectors on the **[[Motherboard]]** that allow additional hardware components to be installed and communicate with the system.

They are used to upgrade or add new functionality to a computer.

- Allow hardware upgrades and expansion
- Provide high-speed data transfer between devices and the system
- Connect critical components directly to the CPU through the motherboard
#### Common devices using PCIe slots

- [[GPU]] 
- [[NIC]]
- Sound cards
- Wi-Fi/Bluetooth cards
- High-speed storage controllers

#### How it works?

Expansion Card → PCIe Slot → Motherboard → CPU

#### PCIe lanes (simple idea)
More lanes = faster data transfer

**Common configurations:**
- x1 → small devices
- x4 → storage devices
- x8 / x16 → GPUs and high-performance cards

In short, PCIe expansion slots are high-speed motherboard connectors used to install and connect additional hardware components like GPUs, storage devices, and network cards.
