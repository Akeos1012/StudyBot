**Storage Devices** – Storage devices are hardware components that **store data, programs, and files permanently or long-term** so they can be accessed even when the computer is turned off.

They work with the **[[CPU]]** and **[[RAM]]** to keep data that needs to be saved beyond active processing.

**Storage devices hold different types of data:**
- Documents and text files
- Images and videos
- Software and applications
- System files (Operating System data)
- Game data and settings

____
### Types of Storage Devices

**Primary Storage (Directly used by CPU):**

[[RAM]] – temporary working memory
[[Cache Memory]] – ultra-fast CPU memory
[[ROM]] – permanent startup instructions

#### Secondary Storage (Long-term storage)

Used to permanently store files and applications:

- [[HDD]] – mechanical storage using spinning disks
- [[SSD]] – faster storage using flash memory
- [[External Storage Devices]] – USB drives, external HDD/SSD
- [[Optical Discs]] – CDs, DVDs, Blu-ray
- [[Cloud Storage]] – online storage accessed via the [[Internet]]

___
### Units of Storage (Data Measurement)

Data in computers is measured in binary-based units:

- **Bit (b)** – smallest unit (0 or 1)
- **Byte (B)** – 8 bits
#### Standard storage units:

- **1 KB (Kilobyte)** = 1,024 Bytes
- **1 MB (Megabyte)** = 1,024 KB
- **1 GB (Gigabyte)** = 1,024 MB
- **1 TB (Terabyte)** = 1,024 GB
- **1 PB (Petabyte)** = 1,024 TB

#### How it works? 

Storage → RAM → CPU → Processing → Output

- Data is first stored permanently in storage
- When needed, it is loaded into RAM
- CPU processes it

In short, storage devices are hardware components that permanently store data and programs, using different technologies like HDD, SSD, and cloud systems, measured in units from bits to petabytes.
