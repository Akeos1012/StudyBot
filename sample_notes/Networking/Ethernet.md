**Ethernet** – Ethernet is a **wired networking technology** used to connect computers and devices within a [[network]] so they can communicate and access the internet.

It sends data through physical cables instead of wireless signals. Ethernet is a wired method of connecting devices to a network and transferring data.

- Connects devices to local networks ([[LAN]])
- Enables internet access through cables
- Transfers data between devices
- Provides stable and reliable communication

#### Main components of Ethernet

- [[NIC]] → sends and receives network data
- Ethernet Cable → physical connection medium
- [[Router]] / [[Switch]] → directs network traffic

#### Common Ethernet speeds

- **10 Mbps** → older networks
- **100 Mbps** → Fast Ethernet
- **1 Gbps** → Gigabit Ethernet
- **10+ Gbps** → high-performance networks

| Ethernet              | Wi-Fi                       |
| --------------------- | --------------------------- |
| Uses cables           | Uses radio signals          |
| Usually lower latency | More convenient             |
| More stable           | More mobile                 |
| Less interference     | Can be affected by distance |

In short, Ethernet is a wired networking technology that allows devices to communicate and access networks using physical cables for fast and stable data transfer.
