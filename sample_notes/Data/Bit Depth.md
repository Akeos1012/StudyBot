**Bit Depth** – The **number of bits used to represent each audio sample in digital audio, which determines the resolution and quality of the sound**.

- Audio quality (clarity and precision)
- Dynamic range (difference between soft and loud sounds)
- Noise level (higher bit depth = less noise)
- File size of audio recordings

#### Common bit depths

- **8-bit** → low quality, more noise
- **16-bit** → CD quality audio
- **24-bit** → professional recording
- **32-bit float** → high precision audio editing

#### How it works

Each audio sample is stored as a binary number:

```
More bits = more possible values = smoother sound
```

Example:

- 8-bit → 256 levels
- 16-bit → 65,536 levels
- 24-bit → 16 million+ levels

#### Simple analogy

Bit depth is like:

the number of shades of color in a photo — more shades = smoother, more realistic image

In short, Bit depth is the number of bits used to represent each audio sample in digital audio, determining the resolution, dynamic range, and quality of the sound.
