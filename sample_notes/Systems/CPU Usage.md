**CPU Usage** –  The **percentage of the Central Processing Unit’s processing power currently being used by programs, processes, and the operating system at a given time**.

- Measures how much [[CPU]] is being used at a specific moment
- Reflects the load from running applications and Processes
- Increases when heavy tasks like gaming, rendering, or compiling code are running
- Managed by the [[Operating System]] through process scheduling
- Affects system speed, responsiveness, and performance
- Monitored using system tools like Task Manager or system monitors

**Example:**
- 10% CPU usage → system is mostly idle
- 50% CPU usage → moderate workload (apps running)
- 100% CPU usage → CPU is fully busy and may slow down

#### How System Works?

Processes → CPU Scheduler → CPU Execution → Resource Consumption → CPU Usage Display

In short, CPU Usage is the percentage of processing power currently being used by the CPU to execute system tasks, applications, and processes at a given time.
