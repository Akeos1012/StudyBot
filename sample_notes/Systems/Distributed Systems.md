**Distributed Systems** – A distributed system is a **group of independent computers connected through a network that work together and appear to users as a single system**.

- Connects multiple [[Computer]] systems
- Shares [[Data]] and Resources over a [[Network]]
- Splits tasks across machines
- Improves performance and reliability
- Handles large-scale workloads

#### Key characteristics

- [[Scalability]] → can add more machines
- [[Concurrency]] → many tasks at once
- [[Fault Tolerance]] → system still works if one fails
- Transparency → appears as one system
- [[Resource Sharing]] → shared computing power

#### Where it is used

- [[Cloud Computing]] platforms
- Google Search infrastructure
- Social Media systems
- Online Banking systems
- [[Big Data]] processing systems

#### Simple analogy

A distributed system is like:

many delivery drivers working together to complete one big delivery network across a city

Each driver is independent, but they all contribute to one system.

In short, A Distributed System is a collection of independent computers connected through a network that work together to perform tasks and appear as a single unified system to users.
