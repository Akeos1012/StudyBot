**[[ROM]]** – (Read-Only Memory) is a type of computer memory that **stores permanent instructions required to start and operate a computer system**.

![[Pasted image 20260617233446.png]]

It keeps important data even when the computer is turned off.

- System startup instructions (boot process)
- Firmware (basic hardware control software)
- BIOS / UEFI system code
#### How it works? 

ROM → 
runs startup instructions → 
loads OS into RAM → 
CPU begins processing

**ROM tells the computer:**
- how to check hardware
- how to start the operating system
- how to initialize basic system functions

**ROM is non-volatile memory, meaning:**
- Power OFF → data is still stored
- Data is permanent or rarely changed

In short, ROM is permanent memory that stores essential startup instructions needed for the computer to boot and initialize hardware.
