**Encryption** – Encryption is the **process of converting readable data (plaintext) into an unreadable format (ciphertext) using an algorithm and a key, so that only authorized users can access it**.

- Protects [[Data]] from unauthorized access
- Uses algorithms and [[Cryptography]] techniques
- Requires a key (secret value) for locking and unlocking data
- Common in [[Security]] systems and secure communication
- Used in messaging, banking, and online transactions
- Works together with [[Decryption]] to restore original data

Encryption is the process of converting readable data into an unreadable format using algorithms and keys to protect it from unauthorized access.
