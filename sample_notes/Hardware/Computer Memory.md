**[[Computer Memory]]** – Computer memory refers to the **hardware components used to store data and instructions either temporarily or permanently for use by the CPU during processing**. It enables the computer to hold information that is actively being used or will be accessed during execution.

- Stores data and instructions for the [[CPU]] to process
- Includes both temporary memory like [[RAM]] and long-term storage like [[ROM]] and [[Storage Devices]]
- Enables fast access to active data during program execution
- Works closely with [[Cache Memory]] to speed up processing
- Supports system operations such as the [[Boot Process]] and program execution
- Organized using [[Memory Address]] for data retrieval and storage

In short, computer memory is a hardware system that stores data and instructions for fast access by the CPU during processing and execution.
