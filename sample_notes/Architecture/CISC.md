**CISC ** – (Complex Instruction Set Computer) is a **CPU design architecture** where the processor is built to execute **complex instructions that can perform multiple low-level operations in a single instruction**. Instead of relying on many simple instructions, CISC aims to reduce the number of instructions per program by making each instruction more powerful and versatile.

A processor design philosophy where instructions can vary in length and complexity, often allowing a single instruction to handle tasks such as memory access, arithmetic operations, and control flow at once. This reduces the number of instructions needed but increases internal CPU complexity.

CISC is one of the major [[CPU Architecture]] models and is commonly used in general-purpose computing systems where flexibility and compatibility are important.
