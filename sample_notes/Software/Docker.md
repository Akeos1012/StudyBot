**Docker** –  is a **platform that allows developers to package applications and all their dependencies into lightweight, portable containers that can run consistently across different environments**.

- Packages [[Software]] into containers
- Includes everything the app needs (code, libraries, settings)
- Runs consistently across different systems
- Uses [[Operating System]] level virtualization
- Improves deployment in [[Cloud Computing]] and servers

#### What is a container?

A container is:

- a lightweight isolated environment
- runs an application independently
- shares the same [[Kernel]] but stays separated from other apps

#### Why Docker is used

- easy deployment
- fast startup
- consistent environments
- scalable applications
- works well with [[DevOps]] workflows

In short, Docker is a platform that packages applications into containers so they can run consistently and reliably across different computing environments.
