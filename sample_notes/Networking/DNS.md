**DNS** – (Domain Name System) a **system that translates human-readable domain names (like google.com) into IP addresses that computers use to locate each other on a network**.

- Converts domain names → [[IP Address]]
- Helps browsers find [[Server]] locations
- Enables access to websites using names instead of numbers
- Routes traffic across the [[Internet]]
- Supports email and online services

#### How it Works?

When you type:

```
google.com
```

DNS returns something like:

```
142.250.190.14
```

Then your browser connects to that server.

#### DNS structure

- Root DNS servers
- Top-Level Domain (TLD) servers (.com, .org, .ph)
- Authoritative DNS servers
- Local DNS resolver (your ISP or router)

#### Simple analogy

DNS is like:

a contact list that turns a person’s name into their phone number so you can call them
