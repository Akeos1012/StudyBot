**[[RAM]]** – (Random Access Memory) The computer’s **temporary working memory** used to store data and instructions that are currently being used by the **[[CPU]]**.

![[Pasted image 20260617233124.png]]

It allows the CPU to access information quickly while programs are running.

- Running applications
- Open files
- Operating system data
- Instructions currently being executed

**RAM is volatile memory, meaning:**
- Power ON → data stays available
- Power OFF → data is erased

**More RAM usually allows:**
- Better multitasking
- More programs open at once
- Reduced loading from storage
#### How it works?

Storage (SSD/HDD)  
↓  
RAM  
↓  
CPU

In short, RAM is fast temporary memory that stores active data and instructions so the CPU can quickly process tasks.
