**Data Bus** – A data bus is a **set of electrical pathways in a computer system that carries actual data between the CPU, memory, and other hardware components**. It is one of the main parts of the [[System Bus]].

- Transfers actual binary data between [[CPU]], [[RAM]], and other devices
- Works alongside [[Address Bus]] and [[Control Bus]]
- Direction can be **bi-directional** (data moves both ways)
- Width of the data bus affects how much data can be transferred at once ([[Bus Width]])
- Speed of transfer is influenced by [[Bus Speed]]
- Essential for communication inside computer hardware systems

A physical part of the system bus responsible for transferring actual data between computer components.
