**Data** - refers to raw facts, symbols, or values that are **collected but not yet interpreted or organized**. It has no meaning on its own until it is processed.

In Computing, data is the **lowest level of digital representation**, stored and transmitted as **binary (0s and 1s)**. All computer operations start from data.

- Raw and unprocessed
- Can be numbers, text, images, audio, or signals
- Has no context by itself
- Used as input for processing

**Examples:**
- `85, 90, 78` (test scores)
- `01001000` (binary data)
- Temperature readings: `30°C, 32°C`
- User input: keyboard or mouse actions

___
# Data vs Information

Data becomes meaningful only after it is **processed, organized, or analyzed**. Once it gains context and purpose, it is called **information**.

**Simple idea:**
- **Data = raw material**
- **Information = processed result**

**Example:**
- Data: 85, 90, 78
- Information: Average score = 84.33

### [[Data Flow in Computer Systems]]

Data moves through multiple layers in a computer system:

- [[Input Devices]] → collect data
- [[RAM]] → temporarily stores data
- [[CPU]] → processes data
- [[Storage Devices]] → saves processed data

In short, computers do not “understand” meaning at the start. They only process data. Meaning is created only after processing.
