**Hexadecimal** – Hexadecimal is a **base-16 number system** used in computing to represent binary data in a more compact and readable form.

It uses sixteen symbols: **0–9 and A–F**, where A–F represent values 10–15.

Hexadecimal is a number system that represents data using 16 symbols to simplify binary representation.

- Shortens long [[Binary Code]] values
- Makes memory addresses easier to read
- Common in [[Computer Memory]] representation
- Used in [[Programming Language]] and system-level coding
- Helps represent colors in web systems (e.g., RGB values)

In short, Hexadecimal is a compact number system used in computing to represent binary data more efficiently, especially in memory, programming, and digital systems.
