**Machine Learning** – Machine Learning is a branch of **[[AI]]** that enables systems to **learn patterns from data and improve performance without being explicitly programmed for every rule or instruction**.

Instead of hard-coded logic, the system builds internal models that adjust based on experience from data. Machine Learning is a method where computers learn from data and improve automatically over time.

- [[Dataset]] → source of learning data
- [[AI Model]] → system that learns patterns
- Training → process of learning from data
- Testing → checking accuracy
- [[Algorithms]] → rules that guide learning

#### Types of Machine Learning

**Supervised Learning**
- Uses labeled data
- Learns input → output relationships

**Unsupervised Learning**
- Uses unlabeled data
- Finds hidden patterns or structure

**Reinforcement Learning**
- Learns through rewards and penalties
- Improves through trial and error

#### What it relies on?

- [[Probability]]
- [[Statistics]]
- [[Linear Algebra]]
- [[Optimization]]

#### Where it is used?

- [[Natural Language Processing]]
- [[Computer Vision]]
- [[Speech Recognition]]
- [[Recommendation System]]
- [[Fraud Detection]]

In short, Machine Learning is a subset of AI where systems learn patterns from data, build models, and improve performance over time without being explicitly programmed for each task.
