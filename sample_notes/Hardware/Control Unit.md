**[[Control Unit]]** – (CU) is a part of the **[[CPU]]** that **directs and coordinates all computer operations by managing how instructions are executed**.

The Control Unit is the part of the CPU that tells all components what to do and when to do it.

- Fetches instructions from memory
- Decodes instructions into signals
- Sends control signals to other components
- Coordinates the **[[ALU]]**, registers, and memory
- Manages the order of execution in the **Instruction Cycle**
### Role in the CPU

**The Control Unit works closely with:**
- [[ALU]] → performs calculations
- [[Registers]] → temporarily store data
- Memory → provides instructions and data

#### How it Works?

Memory → Control Unit → ALU / Registers / Memory

It does not perform calculations itself—it **controls everything that performs the work**.

In short, the Control Unit is the part of the CPU that manages and coordinates all operations by controlling how instructions are executed.
