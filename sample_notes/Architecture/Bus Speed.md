**Bus Speed** – Bus speed refers to the **rate at which data is transferred across a system bus per second**, determining how quickly communication happens between components such as the [[CPU]], Memory, and other hardware units.

It represents a key performance characteristic of the system bus that directly affects how fast data moves between system components. While [[Bus Width]] defines _how much data_ can be transferred at once, bus speed defines _how often_ those transfers occur.

Bus speed is a property of the computer’s [[System Bus]] architecture that controls the timing and frequency of data transfer operations inside the system.

#### How System Works?

Data Source → [[CPU]] / Memory → System Bus → Data Transfer (controlled by Bus Width × Bus Speed) → Destination Component


In short, Bus Speed is the rate of data transfer across a system bus, determining how fast information moves between system components during each operation cycle.
