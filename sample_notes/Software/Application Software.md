**Application Software** – Application software refers to programs designed to **perform specific tasks or functions for users**, rather than managing the computer system itself. It runs on top of an [[Operating System]] and uses system resources to deliver user-focused functionality.

It represents the layer of software that directly interacts with users to complete real-world tasks like creating documents, browsing the web, communicating, or editing media. 

Application software is software designed to help users perform specific tasks on a computer or device.

#### How System Works?

User → Application Software → Operating System → Hardware
