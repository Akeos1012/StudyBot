Boolean - A data type that can only have **two possible values: TRUE or FALSE** (or in computers, **1 or 0**).

It is named after **George Boole**, who developed Boolean algebra, the foundation of digital logic used in computing.

**Example:**
- IF (5 > 3) → TRUE  
- IF (2 == 10) → FALSE

In short, a Boolean is a [[Data]] type that represents logical values: TRUE or FALSE, forming the basis of decision-making in computing.
