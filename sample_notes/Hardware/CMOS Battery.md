**CMOS Battery** – A CMOS battery is a small coin-cell battery on the **[[Motherboard]]** that provides power to store essential system settings when the computer is turned off.

A CMOS battery keeps the computer’s basic system settings saved even when the power is off.

- Keeps BIOS/UEFI settings stored
- Maintains system time and date
- Preserves hardware configuration settings
- Ensures the system can boot properly after shutdown

#### How it Works? 

Power OFF → CMOS Battery keeps settings alive → Power ON → BIOS uses saved settings

Without it, the computer would “forget” basic settings every time it is turned off.

#### What Happens if it Dies?

- Incorrect date and time
- BIOS settings reset to default
- Boot issues or warnings on startup

#### What it Supports?

- [[BIOS]] / [[UEFI]] settings
- System clock (time and date)
- Hardware boot configuration

In short, a CMOS battery is a small battery that preserves BIOS settings and system time when the computer is powered off.
