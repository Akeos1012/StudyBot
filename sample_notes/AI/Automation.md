**Automation** - The use of **technology, systems, or software to perform tasks with minimal or no human intervention**, where processes are executed automatically based on predefined rules, logic, or triggers.

It is built by combining software instructions, hardware control systems, and sometimes intelligent decision-making systems to reduce manual work and increase consistency. Automation is when machines or software handle tasks automatically instead of humans doing them manually.

#### How it Works? 

Trigger → Rule/Logic → System Execution → Output → Feedback (optional)
