**[[Data Cleaning]]** – Data cleaning is the **process of identifying, correcting, and removing errors, inconsistencies, duplicates, and inaccurate information from raw data to improve its quality before analysis or processing**.

- Fixes or removes incorrect [[Data]] entries
- Handles missing values (e.g., blanks, nulls)
- Removes duplicate records
- Standardizes formats (dates, names, units)
- Improves accuracy for [[Data Analysis]] and [[Machine Learning]]
- Common step in [[Data Pipeline]] and [[Data Science]] workflows

Data cleaning is a preparation step that ensures raw data is accurate, consistent, and usable for further processing and analysis.

#### How System Works?

Raw Data → Cleaning → Transformation → Analysis → Insights

In short, data cleaning is the process of fixing and improving raw data so it becomes accurate, consistent, and ready for analysis or machine learning.
