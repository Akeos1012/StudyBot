**Antivirus Software** – A **security program designed to detect, prevent, and remove malicious software (malware) such as viruses, worms, trojans, and spyware from a computer system**.

#### What it does?

- Scans files and programs for threats
- Detects [[Malware]] and suspicious behavior
- Removes or quarantines infected files
- Protects [[Operating System]] and [[Data]]
- Monitors real-time system activity
- Updates virus definitions to handle new threats

#### How it works?

1. Scans files and memory
2. Compares them with known threat signatures
3. Uses behavior analysis for unknown threats
4. Blocks or removes harmful activity

In short, Antivirus Software is a security program that detects, prevents, and removes malware to protect a computer system and its data from malicious attacks.
