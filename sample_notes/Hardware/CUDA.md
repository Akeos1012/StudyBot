**CUDA** –  (Compute Unified Device Architecture) is a **parallel computing platform and programming model developed by NVIDIA that allows developers to use the GPU (Graphics Processing Unit) for general-purpose computing, not just graphics rendering**.

- Enables [[GPU]] to perform massive parallel processing tasks
- Developed by [[NVIDIA]] for high-performance computing
- Used in [[AI]], [[Machine Learning]], scientific simulations, and graphics processing
- Works alongside [[CPU]] to accelerate heavy computations
- Uses programming languages like [[C++]] with CUDA extensions
- Improves performance by splitting tasks across thousands of GPU cores

CUDA is a computing framework that allows software to run complex calculations on GPUs instead of relying only on the CPU, significantly increasing processing speed for parallel tasks.
