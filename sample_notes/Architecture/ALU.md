**ALU** - (Arithmetic Logic Unit) The ALU is the part of the CPU that <u>handles math operations and logic comparisons</u>. 

- It enables decision-making in programs
- It is responsible for all **computation inside the CPU**
- It is essential for everything from simple apps to complex systems like AI
- A core part of the CPU that performs all **mathematical calculations and logical decisions**

**Types of ALU Operations:**
- [[Arithmetic Operations]]
- [[Logical Operations]]
- [[Comparison Operations]]
- [[Shift Operations]]

___
#### Role of the ALU in the CPU

**The ALU works closely with:**
- [[Registers]] → temporary storage for input/output data
- [[Control Unit]] → sends instructions and control signals
- [[RAM]] → provides data needed for operations

**Simple flow:**
Data → Registers → ALU → Result → Registers/Memory

___
#### What the ALU does? 

**The ALU performs two main functions:**

1. **Mathematical Processing** - Handles all numerical calculations required by programs.

2. **Logical Decision Processing** - Handles comparisons and logical conditions used in decision-making.

___
#### How it works in real execution?

When a program runs:
1. The [[Control Unit]] sends an instruction to the ALU
2. The ALU retrieves data from [[Registers]]
3. The ALU performs the required operation
4. The result is stored back into registers or memory

**Example:**
5 + 3 → ALU processes → result = 8
