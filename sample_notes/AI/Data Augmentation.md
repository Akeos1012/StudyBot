**[[Data Augmentation]]** – Data augmentation is a **technique used in machine learning where existing data is artificially expanded by creating modified versions of it, in order to improve model performance and reduce overfitting**.

It is widely used when there is limited training data, especially in [[Machine Learning]] and [[Deep Learning]].

- Creates new training samples from existing [[Data]]
- Improves model generalization and reduces [[Overfitting]]
- Common in image, text, and audio datasets
- Used heavily in [[AI]] training pipelines
- Helps models learn variations and patterns more effectively
- Does not require collecting new real-world data

Data augmentation is a machine learning technique that increases dataset size by generating modified versions of existing data to improve model accuracy and robustness.

#### Technique Examples

- Image: rotation, flipping, cropping
- Text: synonym replacement, paraphrasing
- Audio: noise addition, speed variation

#### How System Works?

Original Data → Transformation → Augmented Data → Training Model → Improved Performance

In short, data augmentation is a machine learning technique that increases training data by creating modified versions of existing data to improve AI model accuracy and generalization.
