**[[GPU]]** – (Graphics Processing Unit) A specialized processor designed to handle **graphics rendering and parallel data processing** efficiently.

![[Pasted image 20260617233819.png]]

A GPU is a processor that handles graphics and performs many calculations at the same time. It works alongside the **[[CPU]]** to display images, videos, and visual effects on a screen.

- Renders images, videos, and 3D graphics
- Processes visual effects in games and applications
- Accelerates parallel computations (many tasks at once)
- Helps with [[AI]], simulations, and data-heavy processing

#### How it works?

**Instead of processing tasks one-by-one like a CPU, the GPU uses massive parallel processing:**

CPU → sends tasks  
GPU → processes thousands of small operations simultaneously  
Display → shows final image/video

#### Where it is used?

- Video games (graphics rendering)
- Video editing and animation
- AI and machine learning
- 3D modeling and simulations

A GPU is a specialized processor focused on **parallel computation and graphics rendering**, working alongside the CPU to accelerate visual and data-heavy tasks.
