**Data Structure** – A **way of organizing, storing, and managing data in a computer so it can be accessed and modified efficiently**.

![[Pasted image 20260623235217.png]]

It defines how data is arranged in memory and how operations like searching, inserting, and deleting are performed.

- Organizes [[Data]] for efficient use and processing
- Improves performance of algorithms in Programming
- Used in [[Software]] development and system design
- Includes structures like arrays, linked lists, stacks, queues, trees, and graphs
- Works closely with [[Algorithms]] design and optimization
- Stored and manipulated in Memory during program execution

A data structure is a method of organizing data in a computer so it can be efficiently accessed, updated, and managed.
