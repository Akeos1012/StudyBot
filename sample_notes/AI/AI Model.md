**AI Model** – An AI model is a **trained mathematical system inside [[AI]] that learns patterns from data and uses them to make predictions, decisions, or generate outputs**.

It is created during the **Training** process, where the system adjusts internal parameters based on input data until it can produce useful results. An AI model is a trained system that learns from data and produces outputs like predictions or decisions.

#### How it Works? 

[[Dataset]] → Training → Model → Input → Output → Feedback
